# Meteor 命令的操作


## 安装meteor
```
$ curl https://install.meteor.com/ | sh
```

## 创建一个项目
```
$ meteor create myapp
```

## 向meteor服务提交
```
$ meteor deploy *.meteor.com
username: smartx
password: ********
```
## 显示所有部署到服务器的域名
```bash
$  meteor list-sites
```

## 部署服务器
所有设置都在客户端这边完成, 然后通这命令去运程到服务器上进行自动安装软件和配置环境.
