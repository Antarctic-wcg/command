## install nodejs

```sh
$ curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash - 
$ sudo apt-get install -y nodejs
```



## install npm

```sh
$ sudo apt-get install npm 
```


```sh
$ sudo npm install hexo-cli -g 
```


## 创建 blog

```sh
 $ hexo init blog
 $ cd blog
 $ npm install 
 $ hexo server
```

可以通过浏览器访问网址.


## 网站进行布置

```sh
$ hexo g
$ hexo d
```
