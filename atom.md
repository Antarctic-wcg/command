# Atom editor
### Install in ubuntu 
```
$ sudo apt-get install build-essential git libgnome-keyring-dev fakeroot
$ sudo add-apt-repository ppa:webupd8team/atom
$ sudo apt-get update
$ sudo apt-get install atom
```

### Configure 

 * `activate-power-mode` 动感插件 `atl + ctrl + o` :打开插件
 * `vim-mode` vim模式
 * `monokai`  高亮显示
